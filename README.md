# PerpusQu
<p>PerpusQu adalah web Application untuk membantu mengelolah perpustakaan dengan skala kecil</p>

## Teknologi yang di gunakan
- Laravel & Livewire
- Database (mysql)

## How to use
1. Clone Repository
``` 
git clone https://github.com/Farriq-mfq/Perpustakaan-Web.git
``` 
2. Import Database perpusqu.sql ke xampp
3. Run laravel dengan comman `` php artisan serve ``
4. Done

## Admin Dashboard

 1. Admin `{localhost}/admin`
 2. Email = admin@admin.com & Password = admin
